# Pandas Project

This repository contains Python scripts and CSV datasets used for practicing
data analysis with the Pandas library.

## Files Included
- pandas1.py – Pandas practice code
- coffee.py – Coffee data analysis
- customer.py – Customer-related data processing
- CSV files – Sample datasets for analysis

## Skills Used
- Python
- Pandas
- Data Cleaning
- CSV Data Handling

## Purpose
This project is created for learning and practice purposes and demonstrates
basic data analysis workflows using Python and Pandas.
